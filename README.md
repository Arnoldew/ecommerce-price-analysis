# E-Commerce Product Price Analysis

## Project Overview

This project analyzes product prices and popularity in an e-commerce marketplace using data collected through web scraping.

## Dataset

The dataset contains product information including:

- product_name
- price
- description
- reviews
- category

## Tools Used

- Python
- Pandas
- BeautifulSoup
- Matplotlib
- Seaborn

## Analysis Performed

- Price distribution analysis
- Category price comparison
- Product popularity analysis
- Price vs reviews relationship

## Key Insights

- Most products fall within the mid-price range.
- Laptop category has the highest average price.
- Lower priced products tend to receive more reviews.
